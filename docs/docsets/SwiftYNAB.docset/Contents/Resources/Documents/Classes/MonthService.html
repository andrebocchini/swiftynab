MonthService

public class MonthService : Service

Provides access to month operations

  • getMonths(budgetId:lastKnowledgeOfServer:completion:)

    Returns a list of budget months.

    • budgetId: The id of the budget (last_used can also be used to specify the last used budget)
    • completion: Completion handler that takes in two parameters. First parameter contains the list of months, and the second parameter contains any errors encountered during retrieval.

                      lastKnowledgeOfServer: Int? = nil,
                      completion: @escaping ([MonthSummary]?, ErrorDetail?) -> Void)
  • getMonth(budgetId:month:completion:)

    Returns a single month.

    • budgetId: The id of the budget (last_used can also be used to specify the last used budget)
    • month: The budget month in ISO format (e.g. 2016-12-01) (current can also be used to specify the current calendar month (UTC))
    • completion: Completion handler that takes in two parameters. First parameter contains the month, and the second parameter contains any errors encountered during retrieval.
